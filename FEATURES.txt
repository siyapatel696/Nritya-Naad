1. Mudra Detection (Bharatanatyam)
   Description:
   Detect hand gestures (mudras) using the user’s webcam. The system should try to identify the gesture and display its name.

Expected Output:

* Webcam opens
* User shows hand gesture
* System displays detected mudra name (or closest match)
* Basic feedback like “Detected” or “Try again”
